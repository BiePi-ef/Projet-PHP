La base de donnée est en MySQL sur Wamp en local, sur le port 3306. Pour que le site marche, il faudra changer les options de connexion dans
./model/bdd.php.
Les scripts pour la création de la base est dans ScriptSQL.

Pour le CSS, j'ai utilisé une librairie en ligne, bootswatch. Elle n'est pas super adapté au site (ne marche pas en fenetré, pas de wraps à la ligne pour le text des messages...).

Pour l'organisation du code, j'ai utilisé le model MVC, que l'on a vu en cours ce mois ci.
